# Olympics Data Analysis Project

## 📌 Project Overview

This project analyzes historical Olympic Games data using Python.
The goal is to explore trends in athlete participation, medal distribution, sports popularity, and country performance across different Olympic years.

The project uses **data analysis and visualization techniques** to extract insights from the Olympic dataset and present them in a clear and understandable way.

---

## 🎯 Objectives

* Analyze Olympic athlete participation over the years
* Identify top medal winning countries
* Explore the most popular sports in the Olympics
* Visualize medal distribution (Gold, Silver, Bronze)
* Understand trends in Olympic data through graphs and charts

---

## 📂 Dataset

The project uses two datasets:

1. **athlete_events.csv**
   Contains information about Olympic athletes such as:

   * Athlete name
   * Gender
   * Age
   * Height and Weight
   * Team
   * Sport
   * Event
   * Medal

2. **noc_regions.csv**
   Contains country information:

   * NOC code
   * Country/Region name

## ▶️ How to Run the Project

1. Install required libraries

```
pip install pandas numpy matplotlib seaborn streamlit
```

2. Run the Jupyter Notebook for analysis.

3. To run the dashboard:

```
streamlit run app/app.py
```
